# Проект "ДЗ 10_1"

## Описание:

Проект "ДЗ 10_1" - это консольное приложение на Python для управления данными банковских операций клиента. Пользователи могут сортировать записи по дате и статусам проведения операций

## Установка:

1. Клонируйте репозиторий:
```
git clone https://https://github.com/SlavaKuul/Lessons
```

2. Установите зависимости:
```
pip install -r requirements.txt
```

## Использование:
Вызовите функцию `get_mask_card_number(user_card: str)`, для наложения маски на номер карты пользователя

Вызовите функцию `get_mask_account(user_account: str)`, для наложения маски на номер счёта пользователя

Вызовите функцию `filter_by_state(date_list: list, state = 'EXECUTED')`, для сортировки записей по статусу, для выбора статуса используйте параметр state

Вызовите функцию `sort_by_date(date_list: list, sort_order = 'True')`, для сортировки записей по дате. Сортировка по-умолчанию от новых записей к старым, любое значение, кроме "True", меняет направление сортировки
